RAILS ERROR SLIDES

Why is error handling so bad?

  • they're written for the developers and programs themselves, not the users who run into them
  • programs can only understand so much about intent
  • we're lazy

How can we fix this?

  • self-correcting errors
  • community support
  • friendlier stack traces
  • writing for our users, not ourselves
